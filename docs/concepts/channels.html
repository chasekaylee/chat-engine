Manage Channels

PubNub creates channels on-the-fly, and you do not incur any additional charges to use one or many in your application. When you create a PubNub application, all messages will be associated with a channel.

Naming channels

Each channel is identified by a unique name. The name may be a string of up to 64 Unicode characters.

Since the text length of the channel name is counted as part of the entire message, it is best to keep the channel name as short as efficiency and utility allows.

The following characters are prohibited in channel and channel-group names: , / \ * :, non-printable ASCII control characters, and Unicode zero.

Joining a channel

Use the subscribe() method to join a channel and start listening to messages. If the channel does not exist, PubNub creates it.

JavaScript
Swift
Java
pubnub.subscribe({
  channels: ['room-1'],
});
pubnub.subscribe()
  .channels(["room-1"])
  .perform()
pubNub.subscribe()
        .channels(Arrays.asList("room-1"))
        .execute();

Joining multiple channels

With channel multiplexing, a client can join multiple channels on a single socket connection. The client listener will receive messages from all subscribed channels.

You can join to up to 50 channels using channel multiplexing. To join more channels, refer to Managing channel groups.

JavaScript
Swift
Java
pubnub.subscribe({
  channels: ['room-1', 'room-2', 'room-3'],
});
pubnub.subscribe()
  .channels(["room-1", "room-2", "room-3"])
  .perform()
pubNub.subscribe()
        .channels(Arrays.asList("room-1", "room-2", "room-3"))
        .execute();

Leaving a channel

Use the unsubscribe() method to leave the channel and stop receiving messages from it.

If Presence is enabled, PubNub generates a leave event when the user unsubscribes from a channel.

JavaScript
Swift
Java
pubnub.unsubscribe({
  channels: ['room-1'],
});
pubnub.unsubscribe()
  .channels(["room-1"])
  .perform()
pubNub.unsubscribe()
        .channels(Arrays.asList("room-1"))
        .execute();

Managing channel groups

A channel group is a collection of individual channels. A channel can be part of multiple channel groups, and a channel group can be shared among multiple subscribers.

Each channel group can contain up to 2,000 channels, and each PubNub instance can subscribe to up to 10 channel groups.

Reference: advanced channel groups design pattern doc

Joining a channel group

Use the pubnub.subscribe() method to join a channel group.

JavaScript
Swift
Java
pubnub.subscribe({
  channelGroups: ['family'],
});
pubnub.subscribe()
  .channelGroups(["family"])
  .perform()
pubNub.subscribe()
        .channelGroups(Arrays.asList("family"))
        .execute();

Adding channels to channel groups

Use the channelGroups.addChannels() method to add channels to a channel group.

JavaScript
Swift
Java
pubnub.channelGroups.addChannels({
  channels: ['son', 'daughter'],
  channelGroup: 'family',
}, (status) => {
  if (status.error) {
    console.log('operation failed w/ status: ', status);
  } else {
    console.log('operation done!');
  }
});
pubnub.stream().add()
  .channels(["son", "daughter"])
  .channelGroup("family")
  .performWithCompletion({ status in
    if status.isError {
      print("operation failed w/ error: \(status.errorData.information)")
    } else {
      print("operation done!")
    }
  })

                        .channels(Arrays.asList("room-1"))
                        .execute();
            }
        }

        @Override
        public void message(PubNub pubnub, PNMessageResult message) {

        }

        @Override
        public void presence(PubNub pubnub, PNPresenceEventResult presence) {
            if (PnUtils.checkPresence(presence, getUuid(), "join", "room-1")) {
                joinSuccess.set(true);
            }
        }
    });

    observerClient.subscribe()
            .channels(Collections.singletonList("room-1"))
            .withPresence()
            .execute();

    Awaitility.await().atMost(TIMEOUT, TimeUnit.SECONDS).untilTrue(joinSuccess);
}

@Test
public void testJoiningMultipleChannels() throws PubNubException, InterruptedException {
    pubNub.subscribe()
            .channels(Arrays.asList("room-1", "room-2", "room-3"))
            .execute();

    TimeUnit.SECONDS.sleep(TIMEOUT);

    PNHereNowResult hereNowResult = pubNub.hereNow()
            .channels(Arrays.asList("room-1", "room-2", "room-3"))
            .includeUUIDs(true)
            .sync();

    List<String> expectedChannels = Arrays.asList("room-1", "room-2", "room-3");

    assertNotNull(hereNowResult);
    assertEquals(expectedChannels.size(), hereNowResult.getTotalChannels());

    List<String> actualChanels = new ArrayList<>();

    for (Map.Entry<String, PNHereNowChannelData> entry : hereNowResult.getChannels().entrySet()) {
        actualChanels.add(entry.getKey());
        boolean member = false;
        for (PNHereNowOccupantData occupant : entry.getValue().getOccupants()) {
            if (occupant.getUuid().equals(pubNub.getConfiguration().getUuid())) {
                member = true;
                break;
            }
        }
        assertTrue(member);
    }

    assertArrayEquals(new List[]{expectedChannels}, new List[]{actualChanels});
}

Listing channels in a channel group

Use the channelGroups.listChannels() method to list all the channels in a channel group.

JavaScript
Swift
Java
pubnub.channelGroups.listChannels({
  channelGroup: 'family',
}, (status, response) => {
  if (status.error) {
    console.log('operation failed w/ error:', status);
    return;
  }

  console.log('listing push channel for device');
  response.channels.forEach((channel) => {
    console.log(channel);
  });
});
pubnub.stream().audit()
  .channelGroup("family")
  .performWithCompletion({ (result, status) in
    if let errorStatus = status {
      print("operation failed w/ error: \(errorStatus.errorData.information)")
      return
    }

    if let auditResults = result {
      print("listing push channel for device")
      for channel in auditResults.data.channels {
        print(channel)
      }
    }
  })
