Here’s what the guide should cover:

  1. Set up your project. This might be a few steps in Swift and Java.
  2. Log in to PubNub Admin Dashboard and retrieve Publish and Subscribe keys. Enable key settings.
  3. Walk through each use case in steps:
    • initialize PubNub
    • subscribe to channel
    • send and receive messages
    • history
    • presence
  4. Link all the code to UIs. Throw it all together.
